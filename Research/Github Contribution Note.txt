## Key CSS Best Practices to Double-Check

* Use Variables: Ensure your themes rely on modern CSS variables (e.g., --bg-color: #ffffff;) rather than writing entirely separate blocks of duplicate CSS layout rules.
* Keep Layouts Separate: Your theme should only change colors, borders, and shadows. Margins, padding, and flex/grid layouts should remain untouched by the theme switch.
