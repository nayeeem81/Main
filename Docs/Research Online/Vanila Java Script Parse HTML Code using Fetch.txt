To read an HTML page, extract content by its ID, and append it to another container div, you can select the elements using document.getElementById() and use appendChild() or append() to move or copy the content.

Depending on your exact use case—whether you are working with elements on the same page or fetching content from an external HTML page—use one of the code examples below.

Scenario 1: Elements are on the SAME Page

If both the source element and the destination container are already on your current page, you can either move the element or clone it so the original stays in place.

Option A: Move the element entirely

Using appendChild() on an existing DOM node automatically detaches it from its original position and inserts it into the new container.

javascript// 1. Select the content element by its ID
const contentToMove = document.getElementById("source-element-id");

// 2. Select the target container div by its ID

const targetContainer = document.getElementById("container-div-id");

// 3. Append the element (this moves it)

if (contentToMove && targetContainer) {
    targetContainer.appendChild(contentToMove);
}

Use code with caution.

Option B: Copy (Clone) the element without removing the original
If you want the source element to remain in its original spot, you must clone it first using .cloneNode(true).

javascript

const originalContent = document.getElementById("source-element-id");
const targetContainer = document.getElementById("container-div-id");

if (originalContent && targetContainer) {

    // Clone the element and all of its children (true)
    const contentClone = originalContent.cloneNode(true);
    
    // Optional: Remove or change the ID of the clone to prevent duplicate IDs on the page
    contentClone.removeAttribute("id"); 
    
    // Append the clone to the container
    targetContainer.appendChild(contentClone);
}

Use code with caution.

Scenario 2: Fetching content from an EXTERNAL HTML Page

If you need to read a completely separate HTML file (e.g., page.html), extract a specific ID from it, and append it to your current page's container, use the JavaScript Fetch API combined with DOMParser.

javascript

async function loadExternalContent() {
    const targetContainer = document.getElementById("container-div-id");

    const url = "path/to/your/page.html"; 
    // The external HTML page URL

    try {

        // 1. Fetch the external HTML file source code as text

        const response = await fetch(url);

        const htmlText = await response.text();

        // 2. Parse the text string into a temporary HTML Document

        const parser = new DOMParser();

        const doc = parser.parseFromString(htmlText, "text/html");

        // 3. Find the specific element by ID within the parsed document

        const externalContent = doc.getElementById("source-element-id");

        // 4. Append it to your container div

        if (externalContent && targetContainer) {

            // Use appendChild to inject the element directly into your container

            targetContainer.appendChild(externalContent);

        } else {

            console.error("Target container or source element ID not found.");

        }
    } catch (error) {

        console.error("Error fetching or parsing the HTML page:", error);

    }
}

// Execute the function

loadExternalContent();

Use code with caution.

Quick Method Reference:

1. document.getElementById(): Finds a specific HTML element using its unique id attribute.

2. parentNode.appendChild(childNode): Inserts a DOM node at the very end of the specified parent container.

3. DOMParser(): A native web API used to convert a raw string of HTML text into a navigable DOM object.
