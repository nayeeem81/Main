## 🚀 **Quick Start (3 Steps)**

### **1. Install NuGet Packages**
```bash
cd Main.WebAppCore
dotnet add package Serilog Serilog.AspNetCore Serilog.Sinks.Console Serilog.Sinks.File Serilog.Formatting.Compact
```

### **2. Create & Apply Migration**
```bash
cd Main.Migrator
dotnet ef migrations add AddExceptionLogsTable --project ..\Main.Infrastructure\
dotnet ef database update
```

### **3. Restart Application**
```bash
dotnet run
```

✅ **Logs will appear in**: `{App Root}/Logs/` directory  
✅ **API available at**: `https://yourdomain.com/api/exceptionlogs/`  
✅ **Database tables**: ExceptionLogs (with 6 indexes)
