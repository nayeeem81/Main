### meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1">

This HTML tag is the cornerstone of responsive web design. It tells a mobile browser to match the webpage's width to the device's screen size and to load the page at the standard 100% zoom level, ensuring the site displays perfectly across all phones and tablets.

How It Works

Without this tag, mobile devices assume a website is built for a wide desktop screen and will zoom out to fit everything, forcing users to squint and pinch-to-zoom. 

This tag forces the browser to adapt to the physical screen.Here is what the two specific parts of the tag do:

width=device-width: 
This matches the webpage's layout width to the actual screen width of the device being used (e.g., an iPhone or an Android).

initial-scale=1: This sets the zoom level to 100% the moment the page loads. It prevents the browser from automatically zooming in or out.

Where to Use 
ItThis snippet must be placed inside the <head> section of your HTML document on every webpage you build.
